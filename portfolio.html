My Portfolio


Hello, I'm Aaron Lurie! I am a current senior at UCF majoring in Digital Media Game Design. In the last four years, I have worked on a lot of different projects and specialize in level design as well as menu UI. Provided below are some examples of some of the projects I've worked on throughout the past few years. Click the buttons to cycle through various screenshots and pictures of my work!

CROSSWORD DATABASE

Creator/Designer

Since 2022, I have been creating crossword puzzles in my free time. In early 2024, I created a website from scratch using HTML code to host all my puzzles for free. I create the puzzles using a website called Crosserville and upload them to a site called Squares.io, which allows me to embed them in my website. My website is still regularly updated with new crosswords being added.

You can view the website here.

ROGUESHAMBO - Team RPS

Lead Coder

I was the primary coder for this project and worked closely alongside the designers and producer to make all the game's mechanics work properly. I coded all of the scripts used in the game in C# and annotated all my code to make it easier for designers to edit and manipulate it.

You can view and download the game on itch.io here.

THERMOFLUX - Forward Slash Studios

Lead Designer

I was the lead designer for this project and headed most parts of level whiteboxing. I oversaw the creation of each of the game's levels and was crucial in playtesting to make sure levels worked properly. I also created the game's menus and menu UI.

You can view and download the game on itch.io here.

DON'T WAKE UP - GDK Spooky Jam 2024

Designer

I was one of two designers for this game created during a 48 hour game jam. Preston Ferrer and I worked together to design the layout of the house you traverse. After getting the assets, I created the house in Unity. I had to get creative in some areas as I wasn't given all the assets we had needed, so in many places I had to improvise and "create" new assets by combining different assets together.

You can view and download the game on itch.io here.
